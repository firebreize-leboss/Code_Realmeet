Dans activity-detail.tsx, le bouton Rejoindre est un TouchableOpacity qui utilise actionLoading pour afficher un ActivityIndicator mais n'a PAS de prop disabled. C'est pour ça que le double-tap passe malgré le navigationLockRef — le bouton reste physiquement tappable. Le bouton +1 n'a pas ce problème car il utilise le composant Button qui a disabled={loading || disabled} intégré. Fix simple : ajouter disabled={actionLoading} sur le TouchableOpacity du bouton Rejoindre/Se désinscrire dans le footer. Cherche le TouchableOpacity qui contient le texte 'Rejoindre' / 'Se désinscrire' / 'Complet' / 'Choisir un créneau' et ajoute disabled={actionLoading || isFull && !isJoined} dessus. Garde le navigationLockRef en sécurité supplémentaire.

# Prompt Claude Code — Clarifier l'affichage des stats sur les chips de créneaux

## Contexte

Dans le fichier `/home/ubuntu/code_realmeet/realmeet-checkin/public/staff/index.html`, les chips de créneaux affichent actuellement une ligne de stats ambiguë : `0 / 1 inscrits` (qui signifie 0 checked-in sur 1 inscrit, mais les utilisateurs confondent avec 0 inscrits sur 1 place).

## Modification à faire

Dans la fonction `loadTodaySlots()`, remplacer la ligne de stats dans chaque chip par **deux lignes distinctes** :

Actuellement :
```html
<span class="slot-chip-stats">
  <span class="done">${s.stats.checkedIn}</span> / ${s.stats.total} inscrits
</span>
```

Remplacer par :
```html
<span class="slot-chip-stats">
  📋 ${s.stats.total} inscrit${s.stats.total > 1 ? 's' : ''}<br>
  <span class="done">✅ ${s.stats.checkedIn} enregistré${s.stats.checkedIn > 1 ? 's' : ''}</span>
</span>
```
